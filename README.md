**🚀 Enterprise DevOps Platform

📌 Overview**

This project demonstrates a complete DevOps pipeline for deploying a containerized application to Kubernetes using modern DevOps tools.

**⚙️ Architecture**
CI/CD: GitHub Actions
Containerization: Docker
Orchestration: Kubernetes (Minikube)
Deployment: Helm
Monitoring: Readiness & Liveness Probes

**🔄 CI/CD Pipeline**
Code pushed to GitHub
GitHub Actions builds Docker image
Image is pushed to registry
Application deployed to Kubernetes
Health checks validate deployment

**🧪 Features**
Automated build and deployment
Containerized Flask application
Kubernetes deployments with Helm
Health monitoring with probes
Troubleshooting scenarios (ImagePullBackOff, connectivity issues)

**🛠️ Technologies**
Docker, Kubernetes, Helm, GitHub Actions, Flask, Linux, Bash
